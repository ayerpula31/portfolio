Smart Resume Screening Web App

Developed a web application that ranks resumes against job descriptions using NLP-based matching to support faster shortlisting.

  • Implemented text preprocessing and similarity scoring with Python and Scikit-learn.
  • Built REST APIs with FastAPI and connected a simple frontend for file upload and score visualization.
  • Added explainable scoring outputs (matched skills, missing skills) to improve recruiter confidence.
